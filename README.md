## Demo
http://spaform.608d273b.svc.dockerapp.io:8080/

## Deploy
```
$ git clone https://github.com/dsdenes/spaform
$ cd spaform
$ npm install
```

## Test
```
$ npm test
```

## Development server for frontend
````
$ npm start 
```

## Development server for API
````
$ DEBUG=spaform:* src/api/bin/www 
```

## Run in production
```
$ npm run build
$ npm run production
```
